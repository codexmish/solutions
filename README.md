# TypeScript Fundamentals: Problem Solving

This repository contains solutions to 7 core TypeScript problems. The exercises cover essential concepts such as Type Aliases, Interfaces, Generics, Class Inheritance, and Array Manipulation.

## Problems & Solutions

### 1. Filter Even Numbers
A function that takes an array of numbers and returns only the even ones.
- **Key Concepts:** Array filtering, Type annotations.

### 2. Reverse String
A utility function to reverse any given string.
- **Key Concepts:** String methods (`split`, `reverse`, `join`).

### 3. Type Checking (Union Types)
A function that identifies whether an input is a `string` or a `number` using Type Guards.
- **Key Concepts:** Union Types, `typeof` operator.

### 4. Dynamic Property Access (Generics)
A generic function to safely retrieve a value from an object using its key.
- **Key Concepts:** Generics `<T>`, `keyof` operator.

### 5. Book Status Toggle
Using an interface to represent a book and a function to append a "read" status using the spread operator.
- **Key Concepts:** Interfaces, Optional parameters, Object spreading.

### 6. Student Inheritance
Implementation of Class Inheritance where a `Student` subclass extends a `Person` base class.
- **Key Concepts:** Classes, `constructor`, `extends`, `super()`.

### 7. Array Intersection
A function that finds common elements between two numeric arrays.
- **Key Concepts:** Function Type Aliases, `filter`, and `includes`.
